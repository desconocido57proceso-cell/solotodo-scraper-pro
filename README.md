# solotodo-scraper-pro
Automated tool to monitor tech prices in the Chilean market using Selenium and Pandas
# Chilean Tech Market - Python Web Scraper Traducción: # Mercado Tecnológico Chileno - Scraper Web en Python
This tool automates the extraction of laptop prices and technical specifications from SoloTodo (Chile). It is designed to help businesses monitor market trends and competitor pricing efficiently. Traducción: Esta herramienta automatiza la extracción de precios de laptops y especificaciones técnicas de SoloTodo (Chile). Está diseñada para ayudar a las empresas a monitorear las tendencias del mercado y los precios de la competencia de manera eficiente.
## 🛠️ Key Features Traducción: ## 🛠️ Características Clave
- **Dynamic Scraping:** Uses Selenium to navigate through multiple pages and bypass dynamic content. Traducción: - Scraping Dinámico: Usa Selenium para navegar a través de múltiples páginas y saltar contenido dinámico.
- **Advanced Data Extraction:** Implements XPath and Regex for precise identification of RAM, CPU, and prices. Traducción: - Extracción de Datos Avanzada: Implementa XPath y Regex para la identificación precisa de RAM, procesador y precios.
- **Professional Reporting:** Generates clean Excel (.xlsx) files with auto-adjusted columns for executive review. Traducción: - Reportes Profesionales: Genera archivos Excel (.xlsx) limpios con columnas auto-ajustadas para revisión ejecutiva.
## 💻 Technologies Used Traducción: ## 💻 Tecnologías Utilizadas
- Python 3.x, Selenium, Pandas, Regex, Openpyxl.
